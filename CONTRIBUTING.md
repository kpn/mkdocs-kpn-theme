# Contributing

## Prerequisites

Install [uv](https://docs.astral.sh/uv/getting-started/installation/).

## Getting started

Install dependencies:

```sh
uv sync
```

Run the development server:

```sh
uv run mkdocs serve -w .
```

The site will be available at `http://127.0.0.1:8000`.

## Adding dependencies

```sh
uv add <package-name>
```
